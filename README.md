# 📊 DataLens - Auto Insights Generator

## 📘 Project Overview
**DataLens** is a Streamlit-based web application designed to automate insights generation for CSV and Excel datasets.

It allows users to upload data and instantly receive:
- Data previews
- Statistical summaries
- Data quality checks
- Visual insights
- Downloadable HTML reports

This tool helps analysts, students, and professionals quickly understand datasets without writing code.

---

## ⚙️ Features

- 📂 Upload CSV and Excel (.xlsx) files  
- 📑 Select specific sheets from Excel files  
- 🔍 Instant dataset preview  
- 📌 Basic dataset information (rows, columns, missing values)  
- ⚠️ Missing value detection  
- 📊 Statistical summary of numeric data  
- 🔥 Correlation heatmap visualization  
- 📈 Optional distribution plots  
- 🧠 Automated insights:
  - Missing values detection  
  - Duplicate rows detection  
  - Constant columns detection  
  - Skewed data detection  
  - Outlier detection (IQR method)  
- 📄 Downloadable HTML report
